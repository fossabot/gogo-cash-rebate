# gogo-cash-back

Cash rebate calculator for Taishin @GoGo

> 台新 @GoGo 現金回饋計算機

## Installation

Make sure you have [yarn](https://yarnpkg.com/) installed

```
yarn
```

## Usage

`node .`

App will listen on port 9090
